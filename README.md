📌 Project Overview

This project analyzes the famous Titanic dataset to discover survival patterns among passengers.

Using Python libraries such as Pandas, Matplotlib, and Seaborn, the project performs:

Data cleaning

Exploratory Data Analysis (EDA)

Data visualization

Survival pattern analysis

The project answers important questions like:

Who survived more: males or females?

Did passenger class affect survival?

How did age impact survival chances?

📂 Dataset

Dataset Used: Titanic Dataset (Kaggle)

The dataset contains:

Passenger details

Gender

Age

Ticket class

Fare

Survival status
